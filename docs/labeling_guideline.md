# 客户评价文本标注规则

## 1. 标签体系

本项目将客户评价分为五类：

| 标签 | 说明 |
|---|---|
| 正向 | 明显满意、推荐、夸赞、复购意愿 |
| 中性 | 表达平淡，无明显褒贬 |
| 负向 | 明显不满、投诉、差评 |
| 无效 | 空白、乱码、纯数字、返现话术、默认好评等无实际语义文本 |
| 不确定 | 同时包含正负信息，或语义不足以稳定判断 |

## 2. 判断示例

### 正向

- 质量很好，下次还会买
- 物流很快，包装完整
- 性价比不错，推荐

### 中性

- 还可以
- 一般般
- 和描述基本一致

### 负向

- 质量差，不推荐
- 发货太慢
- 包装破损

### 无效

- ？？？
- 11111
- 评价返现
- 默认好评

### 不确定

- 物流很快但是质量一般
- 包装可以但是价格偏贵
- 还没用，先好评

## 3. 标注原则

1. 无效文本不归为负向。
2. 正负信息同时出现时，优先标为不确定并备注原因。
3. 难以判断的样本不强行归类。
4. 标注结果保留备注字段，方便后续复核。
